# WELCOME TO CARAVAN FRONTEND PROJECT

WE USE STRICT ORDER FOR CSS STYLES:

{
   flex,
   flexDirection,
   justifyContent,
   alignItems,
   alignSelf,
   backgroundColor,
   borderRadius,
   gap,
   width,
   minWidth,
   maxWidth,
   height,
   minHeight,
   maxHeight,
   padding,
   paddingTop,
   paddingBottom,
   paddingLeft,
   paddingRight,
   paddingVertical,
   paddingHorizontal,
   margin,
   marginTop,
   marginBottom,
   marginLeft,
   marginRight,
   marginVertical,
   marginHorizontal,
   zIndex,
   fontFamily,
   fontSize,
   color
}

It is important to keep this order when writing css, as it makes easier to debug, compare, and familarity.
